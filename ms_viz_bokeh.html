As you can see, the typical characteristics of a tandem mass spectrum are:

  • High density of the m/z values, a spectrum has dozens or even hundreds of points
  • Substantial differences in the intensity (height) of the signals

It is impossible to put all m/z labels onto the plot, but it is of primary interest for the scientists to see those values. An interactive highlighting of m/z would come very handy!

def create_p(width=800, height=300):
    tooltips = [
        ('m/z','@mz{0.0000}'),
        ('Int','@Intensity')
        ]
    p = figure(
        plot_width=width, plot_height=height,
        title = mainTitle,
        tools = 'xwheel_zoom,xpan,box_zoom,undo,reset',
        tooltips=tooltips
        )
    return p
p = create_p()

p.vbar(
    x = 'mz', top = 'Intensity',
    source = cds,
    color = '#324ea8',# alpha = 0.8,
    width = 0.001
    )

show(p)

But there's a problem with the bar plots: they have constant width. If we set the width to a meaningful value that is based on the actual uncertainty of the m/z measurement, it will be extremely narrow. And the hover tool does not work as we would want it to do!

                data=dfA2[
                    dfA2['Annotation'] == cat

p = create_p()

maxIntens = df2['Intensity'].max()
